FAQ

Where is my progress tracker data stored?

Entries are stored only in your browser using localStorage. There is no backend account or server storage.

Can I delete entries?

Yes. Each saved snapshot has a delete button so you can remove it from localStorage.

Does the tracker report to credit reference agencies?

No. It is a personal planning tool only and does not report anything externally.

What does the improvement summary compare?

It compares your earliest and latest saved snapshots for risk score, CCJ status, defaults, utilisation and electoral roll status.
